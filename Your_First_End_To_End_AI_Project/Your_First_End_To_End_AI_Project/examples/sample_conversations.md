# 💬 Sample Multi-Turn Conversations

## Weather Follow-ups
```
You: What's the weather in London?
Assistant [weather]: 14C, partly cloudy.
You: What about Tokyo?
Assistant [weather]: 28C, clear skies.
You: Which was colder?
Assistant [direct]: London at 14C.
```

## Mixed Tools
```
You: What is 25% of 480?
Assistant [calculator]: 120.
You: Tell me about percentages
Assistant [wikipedia]: A percentage is a fraction of 100...
You: Convert 120 F to C
Assistant [converter]: 48.89C
```
